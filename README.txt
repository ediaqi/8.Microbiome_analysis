# Microbiome Analysis in R

This project contains an example workflow for microbiome data analysis in R using the `phyloseq` package.

## Features
- Alpha diversity (Shannon, Simpson)
- Beta diversity (PCoA using Bray-Curtis distance)
- Relative abundance barplots and violin plots

## File Structure
- `data/`: simulated data files (`asv.tsv`, `taxonomy.tsv`, `metadata.csv`)
- `results/`: generated plots
- `scripts/`: R script for full analysis
